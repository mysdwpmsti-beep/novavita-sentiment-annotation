# NovaVita Co. — NLP Sentiment Annotation Project

## Project Overview
End-to-end text annotation project classifying customer reviews 
for NovaVita Co., a premium health & wellness brand based in Amsterdam.

## Dataset
- **Total reviews annotated:** 50
- **Tool used:** Label Studio (HumanSignal Cloud)
- **Output format:** JSON
- **Date completed:** May 31, 2026

## Label Distribution
| Label | Count | Percentage |
|-------|-------|------------|
| Positive | 15 | 30% |
| Negative | 12 | 24% |
| Neutral | 13 | 26% |
| Mixed | 10 | 20% |

## Skills Demonstrated
- Sentiment classification (4 categories)
- Annotation guideline creation
- JSON dataset output for NLP training
- Quality reporting
